02 데이터를 다루는 사람들
- Data Engineer(=개발자)
  대용량의 데이터를 수집, 관리하고 필요한 데이터를 추출하는 역량
  도구: MySQL, Spark, MapReduce, Airflow, 프로그래밍..

- Data Scientist/ ML Engineer
  대용량 데이터를 추출, 처리가 가능하고 그로부터 인사이트 뽑는 역량
  도구: Apache Spark, Machine Learning, 프로그래밍, 통계학

- Data Annotator

- Data Analyst(=비즈니스 문제 해결, 기획)
  데이터를 통해 '문제'를 해결하는데 집중하는데 필요한 역량
  도구: SQL(데이터 조회/활용), 각종 통계도구, 도메인 지식

<사전 테스트>
데이터 분석의 첫 단계로 어떤 과정을 먼저 수행해야 할까요?
A. 데이터 탐색과 이해

데이터 분석 프로젝트에서 "문제 정의"의 중요성에 대해서 설명하시오.
A. 데이터 분석 프로젝트의 출발점이며 중요한 단계이다. 데이터 분석의 목표와 범위를 명확하게 설정하고, 데이터 수집 및 분석 방법을 결정할 때 도움을 준다.

과적합을 방지하는 방법은?
A. 교차검증

A/B 테스트의 주 목적은?
A. 사용자 경험 향상

A/B테스트에서 유효한 결과를 얻기 위해 무엇이 필요한가?
A. 대조군과 실험군 간의 차이가 있어야 함
  대조군과 실험군이 동일한 데이터셋을 공유해야 함

Tableau '워크시트'와 '대시보드'의 주요 차이점을 무엇인가요?
A. 워크시트는 데이터 연결에 사용되고 대시보드는 시각화에 사용됨

SQL은 무엇의 약자이며, 어떤 역할을 수행하는 언어인가요?
A. Structured Query Language, 구조화된 데이터 검색 언어

SQL에서 "WHERE"절은 무엇을 수행하는데 사용됩니까?
A. 데이터 필터링

파이썬 Numpy 라이브러리의 주요 용도는 무엇인가요?
A. 과학적 계산

파이썬 Matplotlib은 어떤 유형의 라이브러리인가요?
A. 시각화 라이브러리
